## Reading Blocks, Control Flow, and Repetition from Python

In Musi, blocks and matches are expressions when they produce a value; the branch answer matters more than the statement container. For a Python reader, the key topic is blocks, control flow, and repetition.

## Common mistake

Do not keep extra variables only to move a branch result out of a statement block. In Python, this appears often in blocks, control flow, and repetition.
